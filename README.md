# ChurnFlix-AI
# 🎬 Netflix Customer Churn Prediction System

# 📌 Project Overview

Netflix Customer Churn Prediction is a Machine Learning based project that predicts whether a customer is likely to cancel their subscription or continue using the service.

The system analyzes customer behavior, subscription details, engagement patterns, and payment information to identify customers who are at risk of leaving and provides retention suggestions.

# 🎯 Problem Statement

Customer retention is one of the major challenges for subscription-based platforms. Losing customers affects revenue and growth.

This project uses Machine Learning algorithms to predict customer churn and helps businesses take proactive steps to retain customers.


# 🚀 Features

- ✅ Predict whether a customer will churn or stay
- ✅ Calculate churn probability percentage
- ✅ Identify customer risk level (Low/Medium/High)
- ✅ Provide personalized retention suggestions
- ✅ Machine Learning based prediction system
- ✅ Interactive Streamlit Web Application

## 📂 Dataset Information

**Dataset Name:** Netflix Customer Churn Dataset

The dataset contains customer-related information:

- Customer Age
- Gender
- Country
- Subscription Plan
- Subscription Duration
- Monthly Charges
- Watch Time
- Number of Devices Used
- Payment Issues
- Customer Satisfaction Score
- Support Queries
- Churn Status
